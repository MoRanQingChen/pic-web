# pages

> 一个基于 ipfs、wilddog 的无限图床


> 不需要自己搭建后端，静态页即可直接使用

## 进度

> 页面搭建(已完成)


> ipfs上传功能(已完成)


> 野狗云接入(已完成)


> serverWorker去中心化(待办)


> 页面排版优化(待办)

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
